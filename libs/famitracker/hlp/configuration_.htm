Configuration

The configuration window consists of five categories.

1. General

Wrap cursor Screen wraps when the cursor has reached the end or top of the pattern.
Free cursor edit Removes the cursor from the middle of the screen.
Preview wave-files Preview wave files in the import wave to DPCM open file window.
Key repeat Enables key repetition.
Show row numbers in hex Displays the row number in hexadecimal or decimal notation.
Preview next/previous frame Display the next and previous frame in shaded before the start and after the end of current frame.
Don't reset DPCM on note stop Removes DPCM popping when keys are released. May decrease the level of triangle and noise as a result of DPCM not being resetted. Press F12 to do it manually.
Ignore step when moving Ignore the step setting when moving the cursor and only use it when inserting notes
Pattern colors Display the patterns in colors
Delete-key will pull up rows Pull up rows below the deleted row

Track edit style
FastTracker II Step down after an instrument/effect number was entered
Modplug Stays on the same line after an instrument/effects number was entered
Impulse Tracker Behaves like Impulse tracker when entering effects.

Page jump length: number of rows to skip when pressing page up/down.
2. Appearance
This section is for customizing the colors. Choose between a few predefined schemes or set up your own.

 3. MIDI
Input device Choose the device from which you want to record MIDI messages.
Output device Choose a device you want to send MIDI messages to.
Master sync Making the tracker to step down after each sixth's clock tick.
Do not record key releases Skip recording of key releases.
Map MIDI to NES channels Maps MIDI-channel messages to NES-channels, channel 1 will be square 1, channel 2 is square 2 and so on. If not checked, the selected channel in the tracker will receive all messages.
Record velocities Converts key-velocities into volume levels and stores them in the volume column.
Auto arpeggiate chords Plays an arpeggio if more than one key are pressed on the MIDI keyboard.

4. Sound

Device The device that receives the audio
Sample rate The rate of samples in hertz, default is 44.1 kHz.
Sample size Size of samples, default is 16 bit. Change only if needed.
Buffer length The length of the sound buffer, default is 40 ms. Increase if you're experiencing sound problems.
Bass filtering Sets the bass filtering frequence (also known as a high-pass filter). Default is 16 Hz.
Treble filtering Sets the treble filtering frequence and cut off strength. Default is -3 dB and 12 000 Hz.
Volume Volume level

5. Shortcuts

 This section allows you to customize the key shortcuts in the program. (Note that the menus will still display the default settings.)
